# Virtual Shopping Cart

A virtual shopping web application built using Java Servlet and JSP technologies and MySQL for database. This project allows users to create accounts and login to individual accounts. Accounts start with $1000, and users are able to add items to the cart in the shopping page. Users can finalize and purchase items in the cart, and items appear in the inventory. Inventory and balance are stored in the database.
